## Affected Pages/Routes
- `kb sync` — API 同步流程：增量拉取、错误处理、游标管理
- `kb distill` — AI 蒸馏流程：分类/标签/摘要/实体/置信度分流
- `kb review` — Staging 审核：approve/reject 流程
- `kb query` — 知识库搜索：关键词匹配、空结果处理
- `kb status` — 状态面板：各目录统计

## Key Interactions to Verify
- `kb sync` 增量同步：第二次运行只拉取新笔记
- `kb distill` 置信度分流：high → knowledge/, medium/low → staging/
- `kb review` 批准后文件从 staging/ 移到 knowledge/
- `kb query` 搜索 YAML frontmatter tags 和 Markdown body

## Edge Cases
- API auth token 过期时的错误提示
- API rate limit 时的指数退避重试
- Claude API 返回格式错误时的重试
- Claude API 拒绝处理内容时的降级
- 空笔记的跳过逻辑
- Unicode/中文内容的正确处理
- 重复蒸馏的幂等性

## Critical Paths
- 完整流程：sync → distill → query（必须端到端跑通）
- 完整流程含 staging：sync → distill → review → query
- 失败恢复：sync 中断 → 重新 sync 不丢数据不重复
